# Project Name

A web application built as a team project with a clear separation between frontend and backend.

## Tech Stack

### Frontend
- React (Vite)
- React Router
- TypeScript
- Tailwind CSS
- Lucide Icons

### Backend
- Python
- Django
- REST API
- PostgreSQL (or other database used in the project)

### Tooling & Collaboration
- Git & GitHub
- GitHub Issues & Pull Requests
- Branch-based workflow
- ESLint & Prettier

## Project Structure

This project is developed in collaboration:
- Frontend – responsible for UI, routing and client-side logic
- Backend – responsible for API, business logic and database
- QA / Helpdesk – testing, issue tracking and documentation support
